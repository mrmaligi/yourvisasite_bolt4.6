## TECHNICAL REQUIREMENTS

### Components to Use
- @/components/ui (all existing UI components)
- @/components/layout (all layout components)
- @/components/animations (Framer Motion wrappers)
- Recharts for data visualization
- React Query for data fetching
- Zustand for state management

### Patterns to Follow
- Lazy loading: `const Page = lazy(() => import('./Page'))`
- Error boundaries with ErrorBoundary component
- Loading states with Skeleton components
- TypeScript interfaces for all props
- React Query for data fetching
- Zustand for state management
- React Hook Form for forms
- Zod for validation

### Page Features Required
- SEO meta tags (react-helmet-async)
- Responsive design (mobile-first)
- Accessibility (ARIA labels, keyboard nav)
- Consistent spacing (Tailwind spacing scale)
- Toast notifications for actions
- Route protection where applicable
