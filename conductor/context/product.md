# VisaBuild Product Context

## What is VisaBuild?
Australian visa information platform with 3 roles:
- **Users**: Search visas, unlock premium guides ($49), book lawyers
- **Lawyers**: Verified experts, paid consultations
- **Admins**: Manage platform, approve lawyers

## Core Features
1. Visa catalog (100+ Australian visas)
2. Processing time tracker (crowdsourced)
3. Premium content ($49 unlock per visa)
4. Document upload helper (19 categories)
5. Lawyer directory & booking
6. Admin dashboard

## Success Metrics
- 100% of Tier 1-2 visas have premium content
- Document upload works end-to-end
- Lawyer booking flow complete
- All DHA links official and current

## Constraints
- Australia ONLY for MVP
- Test mode Stripe for 5-day sprint
- Supabase Storage for documents
