# Tech Stack

## Frontend
- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Lucide React (icons)
- React Router v6 (routing)

## Backend
- Supabase (BaaS)
  - Auth (GoTrue)
  - PostgreSQL (database)
  - Storage (documents)
  - Edge Functions (Deno)

## Payments
- Stripe (checkout + webhooks)
- Test mode for development

## Deployment
- Vercel (frontend)
- Supabase (backend)
