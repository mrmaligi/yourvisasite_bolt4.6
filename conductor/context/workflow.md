# Development Workflow

## Cycle: 5 Days, 50 Tasks
- 10 tasks per day
- 2-hour status reports via cron
- Auto-merge if tests pass

## Roles
- **Jules**: Implements code (90% of tasks)
- **Ultra**: Verifies, coordinates, merges (10% of tasks)
- **Mk**: Decides priorities, approves scope

## Task States
- PENDING → IN_PROGRESS → REVIEW → DONE

## Rules
1. Spec before implementation
2. Test after each track
3. Small PRs (< 50 files)
4. Official DHA links only
5. Australia visas only
